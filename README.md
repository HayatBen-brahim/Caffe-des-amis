Caffè des amis - Site Web Officiel
​Bienvenue dans le dépôt GitHub de Caffè des amis. Ce projet est une vitrine numérique conçue pour offrir une expérience immersive aux clients de notre café.
​🎯 Objectif du Projet
​L'objectif est de créer une plateforme élégante qui reflète l'ambiance chaleureuse du café tout en facilitant la communication avec nos clients. En tant qu'étudiante en Commerce et Marketing, j'ai conçu ce site pour intégrer des éléments de branding forts et une stratégie de contenu ciblée.
​🛠️ Technologies Utilisées
​HTML5 : Pour la structure sémantique du contenu.
​CSS3 : Pour le design, incluant des dégradés personnalisés (Gradients) et une mise en page moderne.
​Branding : Utilisation de visuels optimisés pour le marketing digital.
​🚀 Fonctionnalités
​Galerie de Produits : Présentation visuelle de nos créations et offres spéciales.
​Section Contact : Informations de localisation et liens vers les réseaux sociaux pour booster l'engagement.
​Responsive Design : Interface adaptée aux différents types d'écrans.
​📈 Aspect Marketing
​Ce site n'est pas seulement du code, c'est un outil de conversion :
​Mise en place de Call to Action (CTA) stratégiques.
​Optimisation de la structure pour le SEO.
​Cohérence visuelle pour renforcer l'image de marque du café.
